# HomeServer
## 购买背景

由于本人只有一台笔记本电脑，需要带去公司工作，家里没有老旧电脑，且想给家里的电视机、路由器、智能家居等设备提供服务，故购入服务器一台。

迫于钱包过于紧张，只能先入手一台 J1900
